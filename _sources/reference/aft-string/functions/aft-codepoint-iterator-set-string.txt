aft_codepoint_iterator_set_string
=================================

.. c:function:: void aft_codepoint_iterator_set_string( \
        AftCodepointIterator* it, AftString* string)

    Set the string for the iterator and position the iterator over the starting
    index.

    The iteration range is over the entire string.

    :param it: the iterator
    :param string: the string
